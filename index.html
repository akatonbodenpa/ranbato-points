Battle Rating Calculator

対戦ゲーム レート変動計算ツール

「連勝数」「自分のポイント」「相手のポイント」を入力すると、勝った場合と負けた場合のポイント変動を即座に表示します。

勝ち 0 獲得ポイント
負け 0 獲得ポイント

計算式

連勝ボーナス倍率 = min(1.2, 1 + (連勝数 - 1) * 0.05)
ポイント差 = 自分のポイント - 相手のポイント

勝ち: 基礎ポイント = max(1, (4000 - ポイント差) / 25)
      獲得ポイント = int(基礎ポイント * 連勝ボーナス倍率)

負け: 基礎ポイント = -(4000 + ポイント差) / 25
      獲得ポイント = int(基礎ポイント)
